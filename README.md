# Role Setiap Account
## - Kasir
- Data Transaksi.
- Laporan Keuangan.
- Laporan Arus Kas.

## - Admin
- Laporan Keuangan.
- Master Data (Kelola Data User).

## - Akuntan
- Akun (Golongan Akun & Kelola Akun).
- Data Transaksi.
- Jurnal Umum.
- Jurnal Penyesuaian.
- Buku Besar.
- Neraca Saldo.
- Laporan Keuangan.

## - Pemilik
- Laporan Keuangan.

- Install Laravel.
- Install Composer.

# Tutorial Menjalankan Websitenya
- Ekstrak file Rar terlebih dahulu.
- Upload file Sql (dbakunku.sql) di localhost.
- Jalankan perintah "php artisan serve" di terminal.
